Математические модели

Модель Достоинства Недостатки
Sorensen
  • Учет динамики глюкагона
  • Удобные коэффициенты для симуляции диабета
  • Отсутствие вариативности параметров
  • Внутривенное введение инсулина
UVa/Padova
  • Одобрено FDA
  • Проработанная модель абсорбции глюкозы
  • Нет учета динамики глюкагона
  • Нет внутрипациентной вариативности
Cambridge
  • Модель физической активности
  • Выборка клинических данных
  • Нет учета динамики глюкагона
  • Слабая модель абсорбции глюкозы

Прототип функции, реализующей модель UVa/Padova

            def Simulation(
              BW_input=80, #вес тела
              D_input=100000, #масса пищи

              Gp0_input=162, #начальный уровень глюкозы в плазме
              meal_duration=0 #длительность приема пищи
